# 💳 Credit Card Fraud Detection System

🚀 A Machine Learning-based web application that detects fraudulent credit card transactions using an interactive Streamlit interface.


## 🌐 Live Demo

👉 https://sg9164233-png-fraud-detection-app-app-ozb27e.streamlit.app


## 📌 Project Overview

This project aims to identify fraudulent transactions using a classification model trained on an imbalanced dataset. The system provides both real-time prediction and batch processing capabilities through a user-friendly web interface.


## 🚀 Features

* 🔍 Real-time transaction prediction
* 📂 Batch prediction using CSV upload
* 📊 Fraud probability score
* 📥 Download prediction results
* ⚖️ Handles imbalanced dataset using class weighting
* 🌐 Deployed on Streamlit Cloud

## 🧠 Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit

## 📊 Model Details

* Algorithm: Logistic Regression
* Handling Imbalance: `class_weight='balanced'`
* Evaluation Metrics:

  * Accuracy
  * Precision
  * Recall (important for fraud detection)
  * F1 Score
